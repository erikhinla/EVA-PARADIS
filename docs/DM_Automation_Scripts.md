## Tracking Parameters

All bridge URLs should include UTM parameters:

- `utm_source`: platform (reddit, instagram)
- `utm_medium`: dm_type (welcome, engagement, story_reply)
- `utm_campaign`: script_id (r1, r2, i1, i2, etc.)
- `utm_content`: variant (a, b, c)

**Example:**
```
[bridge-url]/vip?utm_source=reddit&utm_medium=dm_welcome&utm_campaign=r1&utm_content=a
```
